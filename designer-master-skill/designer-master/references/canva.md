# Canva — Uso Profissional e Avançado

## Canva não é "só para iniciantes"

Canva Pro nas mãos certas é uma máquina de produção. Designers profissionais usam Canva para:
- Produção em escala (dezenas de peças com variações)
- Entrega rápida para clientes (compartilhamento, edição colaborativa)
- Consistência de marca (Brand Kit centralizado)
- Apresentações e decks de alta qualidade
- Materiais de marketing com resultado em minutos

A diferença entre um amador e um profissional no Canva não está nos recursos — está nas **decisões**.

---

## Brand Kit — O Coração do Uso Profissional

### Como Configurar
1. **Brand Hub** (Canva Pro) → adicione:
   - Paleta de cores (HEX, CMYK, RGB)
   - Tipografias da marca (faça upload das fontes se não estiverem no Canva)
   - Logos em todas as versões (principal, símbolo, horizontal, monocromático, branco, preto)
   - Templates aprovados
2. Use os **atalhos de cor da marca** no painel de cor — um clique, cor certa, sempre
3. Crie múltiplos Brand Kits para clientes diferentes (Canva Pro permite vários)

## Recursos Avançados Pouco Explorados

### Magic Resize (Redimensionamento Mágico)
- Crie uma peça → **Redimensionar** → escolha múltiplos formatos simultaneamente
- O Canva ajusta posicionamento automaticamente — mas **sempre revise** cada versão
- Fluxo ideal: crie no maior formato (ex: 1920x1080), depois redimensione para Stories, Post, Banner

### Grades e Frames
- **Frames** (molduras): arraste imagens diretamente para dentro de shapes pré-formatados
- Ideal para criar colagens profissionais, mockups de produto, layouts de portfolio
- Duplo clique no frame para reposicionar/reescalar a imagem dentro

### Transparência e Mistura
- Selecione elemento → no canto superior direito: ícone de xadrez (transparência) → ajuste com slider
- Para efeitos de sobreposição: use a foto de fundo + uma camada colorida com transparência para criar gradientes de cor sobre foto

### Elementos Vetoriais e Customização
- Elementos SVG do Canva podem ser **desmontados** (Ungroup) para editar cores individualmente
- Use "Aplicar a todos" (Apply to all) para trocar cores em múltiplos elementos de uma vez
- Importe SVGs externos para ter elementos totalmente customizáveis

### Grids de Alinhamento e Espaçamento
- Ative **Guias inteligentes** (Smart Guides) nas configurações
- Pressione **Alt** ao mover para ver distâncias entre elementos automaticamente
- Use **Alinhar e Distribuir** (ícone de régua) para distribuição perfeita entre múltiplos elementos

## Tipografia Profissional no Canva

### Combinações que Funcionam Sempre
| Título | Corpo | Estilo |
|--------|-------|--------|
| Playfair Display | Lato | Luxo / Editorial |
| Montserrat Bold | Open Sans | Tech / Corporativo |
| Josefin Sans | Raleway | Minimalista Premium |
| Bebas Neue | Roboto | Impacto / Esportivo |
| Cormorant Garamond | Source Sans Pro | Elegante / Clássico |

### Regras de Ouro de Tipografia no Canva
- Máximo de **2 fontes** por peça (3 em projetos editoriais complexos)
- **Contraste de peso**: se o título é Bold, o corpo deve ser Regular ou Light
- **Contraste de tamanho**: título deve ser pelo menos 2x maior que o texto de suporte
- **Entrelinha (Line Height)**: para textos longos, use 1.4-1.6x o tamanho da fonte
- **Kerning**: em títulos grandes, ajuste o espaçamento entre letras (-10 a -20) para aparência mais profissional

## Cores no Canva como um Profissional

### Extraindo Paleta de uma Imagem
1. Insira a imagem de referência
2. Clique em qualquer elemento de cor → clique no color swatch → **Adicionar cor de design** → use o eyedropper na imagem
3. Isso cria uma paleta contextual para a peça — garante coesão visual total

### HEX Codes para Marcas — Nunca Aproxime
- Sempre insira o HEX exato da marca, nunca selecione "aproximado" visualmente
- Para converter CMYK para HEX (para impressão → digital): use conversor online como colordesigner.io
- Salve as cores exatas no Brand Kit para nunca errar

## Preparando para Impressão no Canva

### O que funciona e o que não funciona
✅ Funciona bem:
- Flyers digitais para impressão doméstica
- Cartazes simples
- Materiais entregues como PDF em RGB

⚠️ Atenção:
- Canva trabalha em **RGB** nativamente — para impressão offset profissional, avise o cliente que pode haver variação de cor
- Para sangria: acesse **Arquivo > Baixar > PDF para Impressão** → ative a opção "Sangria e marcas"
- Resolução de exportação: Canva Pro exporta em alta qualidade (300 DPI equivalente) — use sempre PDF para impressão, nunca PNG para gráfica

### Formatos de Exportação por Finalidade
| Finalidade | Formato |
|-----------|---------|
| Redes sociais | PNG (com fundo) |
| Apresentação | PDF Standard ou PPTX |
| Impressão gráfica | PDF para impressão |
| Animações | GIF ou MP4 |
| Edição posterior (Photoshop) | PNG com fundo transparente |
| Site/Email | JPEG (menor peso) |

## Produção em Escala — Canva Pro

### Bulk Create (Criar em Massa)
1. Crie um template com **variáveis** (ex: {nome}, {cargo}, {foto})
2. Acesso via **Apps > Bulk Create**
3. Suba uma planilha CSV com os dados
4. Canva gera automaticamente todas as variações

**Uso profissional**: Crachás de evento, certificados, posts personalizados por nome, catálogos de produto

### Colaboração com Clientes
- **Compartilhamento com permissão**: envie link de visualização OU edição para o cliente
- **Comentários**: o cliente comenta direto na peça — sem idas e vindas por e-mail
- **Aprovação de marca**: use Brand Templates bloqueados para que a equipe só altere o que pode alterar

---

## O Que Profissionais Nunca Fazem no Canva

- ❌ Usar templates sem personalizar — templates genéricos = marca genérica
- ❌ Deixar elementos do template que não servem "para não precisar deletar"
- ❌ Usar mais de 3 fontes na mesma peça
- ❌ Colocar texto sobre foto sem camada de contraste (retângulo semitransparente ou gradiente)
- ❌ Exportar PNG quando o cliente precisa de PDF para impressão
- ❌ Aproximar cores de marca "no olho" em vez de usar HEX exato
