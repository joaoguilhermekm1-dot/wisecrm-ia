# Branding Estratégico — Da Identidade ao Resultado de Negócio

## O que é uma Marca (além do logo)

Marca não é o que você diz que é. **Marca é o que as pessoas sentem quando pensam em você.**

O logo, as cores e as fontes são a expressão visual dessa sensação. O trabalho do designer estratégico é criar um sistema visual que evoca, de forma consistente, a emoção e a percepção certas no público certo.

---

## O Processo de Criação de Identidade Visual

### Fase 1: Diagnóstico de Marca (Antes de Abrir Qualquer Software)

**Perguntas obrigatórias para o cliente:**

1. **Propósito**: Por que essa empresa existe além de gerar lucro?
2. **Público-alvo**: Quem é o cliente ideal? (Seja específico: "mulheres de 28-40 anos, classe B, urbanas, que valorizam praticidade e estética" — não "todo mundo")
3. **Concorrentes**: Quais são os 3 principais? Como se posicionam visualmente?
4. **Diferenciação**: Por que alguém deveria escolher essa marca e não a concorrente?
5. **Personalidade**: Se a marca fosse uma pessoa, como ela seria? (Escolha 3-5 adjetivos: "confiável, acolhedora, irreverente")
6. **Posicionamento de preço**: Econômico, intermediário, premium ou luxo?
7. **Tom de voz**: Formal? Informal? Técnico? Descontraído?
8. **Referências visuais**: Mostre 5 marcas que você admira (mesmo fora do setor) e diga por quê

### Fase 2: Definição do Território Visual

Com base no diagnóstico, defina:

**Arquétipo de Marca** (Jung aplicado ao branding):
| Arquétipo | Marcas Referência | Características Visuais |
|-----------|------------------|------------------------|
| Herói | Nike, Adidas | Bold, dinâmico, contrastes fortes |
| Sábio | Google, IBM | Clean, azul, tipografia clara |
| Criador | Apple, Lego | Minimalista, inovador, espaço branco |
| Cuidador | Johnson's, Dove | Suave, quente, curvas, tons neutros |
| Bobo da Corte | M&Ms, Skol | Colorido, divertido, irregular |
| Fora-da-lei | Harley, Red Bull | Escuro, agressivo, ângulos |
| Inocente | Coca-Cola, Mc (família) | Alegre, vermelho/amarelo, redondo |
| Governante | Mercedes, Rolex | Dourado, preto, serif, rígido |

### Fase 3: Sistema Visual

#### Logo
- **Versão principal**: logo completo (símbolo + nome)
- **Versão horizontal**: para uso em assinaturas, headers
- **Versão símbolo**: apenas o ícone, para uso em tamanhos pequenos (favicon, perfil)
- **Versão monocromática**: para bordado, gravação, tampografia
- **Versão negativa**: logo branco para uso sobre fundos escuros
- **Área de proteção**: espaço mínimo ao redor do logo sem outros elementos

#### Paleta de Cores
```
Cor Principal (dominante) — HEX + CMYK + RGB + Pantone
Cor Secundária (complemento) — HEX + CMYK + RGB + Pantone
Cor de Destaque/CTA — HEX + CMYK + RGB + Pantone
Neutros: branco (#FFFFFF), preto (#000000), cinza escuro, cinza claro
```

Sempre forneça os valores em todos os sistemas de cor — o cliente vai usar em diferentes contextos.

#### Tipografia
```
Fonte Primária (títulos): [Nome] — Bold, SemiBold
Fonte Secundária (textos): [Nome] — Regular, Light, Italic
Fonte de Apoio (opcional, detalhes): [Nome]
```

**Regra de ouro**: Tipografias com personalidades contrastantes mas complementares. Ex: Serif clássica para título + Sans-serif moderna para corpo = elegância com legibilidade.

#### Elementos Gráficos Complementares
- Padrão/textura de marca
- Ícones customizados
- Fotografia style guide (tipo de imagem, tratamento de cor, o que pode e não pode aparecer)
- Formas orgânicas ou geométricas que complementam o logo

## Brand Guidelines — O Manual da Marca

Um manual profissional deve ter:

1. **Introdução à marca** (missão, visão, valores, personalidade)
2. **Logo e suas versões** (usos corretos + exemplos do que NÃO fazer)
3. **Paleta de cores** (com todos os valores e proporções de uso)
4. **Tipografia** (hierarquia, tamanhos mínimos, combinações)
5. **Elementos gráficos** (padrões, formas, ícones)
6. **Fotografia** (estilo, tratamento, o que evitar)
7. **Aplicações** (mockups realistas: cartão de visita, papelaria, uniforme, embalagem, Instagram)
8. **Usos incorretos** (mostra erros comuns e como não usar a marca)
9. **Contatos** (quem chamar quando tiver dúvida)

## Rebranding — Quando e Como Fazer

### Sinais de que uma Marca Precisa Evoluir
- Parece desatualizada (design de 10+ anos atrás em setor dinâmico)
- Mudança significativa de público, produto ou posicionamento
- Expansão geográfica (o que funciona localmente pode falhar em outro mercado)
- Associação negativa acumulada (crise de imagem)
- Inconsistência visual grave entre pontos de contato

### Tipos de Rebranding
1. **Refresh**: pequena evolução (modernizar fonte, ajustar cores) — mantém reconhecimento
2. **Redesign**: mudança significativa mas com DNA preservado — quando a marca cresceu
3. **Rebranding total**: novo nome, nova identidade — situações extremas

### Como Apresentar Rebranding para o Cliente
1. Mostre o ANTES e DEPOIS lado a lado
2. Explique cada decisão com base em estratégia (não em "achei mais bonito")
3. Apresente em contexto (mockups realistas, não só o logo sozinho)
4. Mostre como resolve o problema identificado no diagnóstico
5. Apresente o rollout (como vai ser a transição: imediata ou gradual)

## Erros Fatais de Branding

| Erro | Por que é fatal |
|------|-----------------|
| Logo não funciona em P&B | Vai ser bordado, gravado, fotocopiado — desaparece |
| Muito detalhe no símbolo | Vira borrão em tamanhos pequenos (favicon, crachá) |
| Copiar a concorrência | A marca desaparece na categoria em vez de se diferenciar |
| Cores sem justificativa | Falta coerência, a marca não comunica personalidade clara |
| Fontes gratuitas genéricas | Arial, Times New Roman = invisibilidade de marca |
| Não testar em fundo escuro E claro | Metade das aplicações vai falhar |
| Criar sem conhecer o público | Design para o gosto do dono, não para a cabeça do cliente |
